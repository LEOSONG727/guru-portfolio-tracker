## 2. 13F 공시 이해

### 13F란?
- 미국 SEC에 보고 의무가 있는 기관투자자(AUM $100M 이상)의 분기별 주식 보유 현황 보고서
- 보유 내역 기준일 이후 45일 이내 제출 의무

### 공시 시점과 보유 기간
| 공시 | 보유 기간 | 제출 마감 | 분석 시점 |
|---|---|---|---|
| Q1 공시 | 전년도 Q4 (10~12월) | 2월 14일 | 2~3월 |
| Q2 공시 | 당해 Q1 (1~3월) | 5월 15일 | 5~6월 |
| Q3 공시 | 당해 Q2 (4~6월) | 8월 14일 | 8~9월 |
| Q4 공시 | 당해 Q3 (7~9월) | 11월 14일 | 11~12월 |

### 시차 고려
- 공시 시점에 이미 보유 기간 종료 후 최대 45일 지남
- 단기 매매자는 이미 포지션이 변경되었을 가능성 높음
- **장기 투자자(최소 6개월 호흡)의 신호가 가장 신뢰도 높음**

## 4. 신호 강도 공식

```
score = 티어1 신규매수(NEW)  × 2
      + 티어1 비중확대(INC)  × 1
      + 티어2 신규매수(NEW)  × 1
      + 티어2 비중확대(INC)  × 0.5
```

### 해석 기준

| 점수 | 신호 강도 | 해석 |
|---|---|---|
| ≥ 6 | ★★★ 최강 | 복수 티어1 동시 신규매수 |
| 4–5 | ★★ 강함 | 티어1 대규모 신규매수 또는 다수 확대 |
| 2–3 | ★ 중간 | 티어1 비중확대 또는 티어2 신규매수 |
| 1 | △ 참고 | 티어2 비중확대만 |

## 5. 추가 고려사항

### 신호 강화 요인
- **동일 종목에 복수 티어1이 동시 신규매수**: 정보 공유 없이 독립적 판단이 일치 → 최강 신호
- **캐피털 그룹 3사가 동시 움직임**: 사실상 동일 그룹이지만 독립 운용이므로 유효
- **대규모 단일 포지션**: 포트폴리오의 1% 이상 신규 매수는 강한 컨빅션

### 신호 약화 요인
- **ETF·인덱스 펀드 매수**: 지수 편입 효과로 인한 기계적 매수 가능
- **옵션(PUT/CALL) 포지션**: 헤지 목적일 수 있어 분석에서 제외
- **같은 운용사 계열사 중복**: 실질적으로 동일한 의사결정이면 하나로 취급

## 6. 기술 구현 참고사항

### RSC 데이터 수집 방법
```bash
# 투자자 목록 (메인 페이지)
curl -s "https://giantsight.com/ko?_rsc=analyse1" \
  -H "RSC: 1" -H "Accept: text/x-component"

# 인기 종목 페이지
curl -s "https://giantsight.com/ko/2026/Q1/popular-stocks?_rsc=popdata1" \
  -H "RSC: 1" -H "Accept: text/x-component"
```
